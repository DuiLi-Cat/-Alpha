Alpha0.3.3版本内容：
1.平衡了仙修与魔修的属性加成 现在仙修偏向于加血 魔修偏向于耐力
2.完善“境界提升丹”的配方
3.现在檀渊珠和魔渊珠通过boss进行掉落 召唤物可在”制作“-”其他“-”其他“进行合成

Alpha0.3.2版本内容：
1.加入货币
2.重新做了“仙丹”，现在提升境界需要“境界提升丹”（可以吃高级丹提升低级，但不能吃低级丹提升到更高境界）
3.加入“魔修”（仙修属性加成比较平均 魔修属性加成比较偏）
4.现在“炼器”等级可以通过看书升级了（暂时不会自然刷新）
